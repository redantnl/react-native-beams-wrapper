# react-native-beams-wrapper

A react native wrapper for Pusher Beams

## Installation

```sh
npm install react-native-beams-wrapper
```

## Usage

```js
import BeamsWrapper from "react-native-beams-wrapper";

// ...

const result = await BeamsWrapper.multiply(3, 7);
```
